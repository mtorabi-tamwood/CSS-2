important commands in git

  • To see the git status run git status
  • To add a file to stage git add Filename
  • To commit changes with message git commit -m "your message"
  • To clone a remote repository git clone linkofrep
  • To change the alias name of remote repositories git remote rename CurrentName RecentName
  • To see the list of remote connections git remote -v
  • To see list of configuration git config --list
  • To add username git config --global user.name GitHubUserName
  • To add email git config --global user.email GitHubEmail
  • To restore a file from stage git restore --staged FileName
  • To discard any changes git resotre FileName
  • To pull repostory from remote repository git pull RemoteAlias BranchName
  • To push changes to remote repostitory git push RemoteAlias BranchName
